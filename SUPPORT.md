## Reporting Issues

### Bug Reports

Please include:

1. Module name and version
2. Terraform/OpenTofu version
3. Provider version
4. Minimal reproduction configuration
5. Expected vs actual behavior
6. Error messages (if any)

### Feature Requests

Please include:

1. Use case description
2. Proposed solution (if any)
3. Alternative solutions considered

## Response Times

- **Bug Reports**: We aim to triage within 7 days
- **Feature Requests**: Reviewed monthly
- **Security Issues**: Within 48 hours (see [SECURITY.md](SECURITY.md))
